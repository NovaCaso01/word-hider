# 단어 가리기 (Word Hider) - SillyTavern Extension

메시지에서 특정 단어나 문장을 가리는 SillyTavern 확장프로그램입니다.

## ✨ 기능

- 🔒 특정 단어/문장을 가리기
- 🎨 **색상**으로 가리기 (직접 색상 선택)
- 😊 **이모지**로 가리기 (하트, 펭귄, 로봇 등)
- ✨ **아스키아트**로 가리기 (구름, 나비, 리본 등)
- 📝 여러 단어를 각각 다른 방식으로 가리기
- 🔘 온/오프 토글 기능

## 📱 사용법

1. 입력창 옆 **마법봉 버튼(✨)** 클릭
2. **"단어 가리기"** 선택
3. 팝업에서:
   - 가릴 단어/문장 입력
   - 가리기 방식 선택 (색상/이모지/아스키아트)
   - **"추가"** 클릭
4. 상단 토글로 전체 기능 온/오프
